cActiveMQ

cActiveMQ properties

Component Family

Messaging

Function

cActiveMQ allows messages to be sent to a JMS Queue or Topic or messages to be consumed from a JMS Queue or Topic. cActiveMQ is based on cJMS.

Purpose

cActiveMQ is used to send messages to, or consume messages from, a JMS Queue or Topic.

 Basic settings

URI/Type

Select the message type, either queue or topic.

 

URI/destination

Type in a name for the JMS queue or topic.

 

Broker/broker URI

Type in the URI of the broker you want to use to send or consume messages.

Advanced settings URI Options Set the optional arguments in the corresponding table. Click [+] as many times as required to add arguments to the table. Then click the corresponding Value field and enter a value. See the site http://camel.apache.org/jms.html for available URI options.

Usage

cActiveMQ can be a start, middle or end component in a route.
