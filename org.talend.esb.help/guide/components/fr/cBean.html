cBean

Propriétés du cBean

Famille de composant

Processing

Fonction

Le composant cBean un bean Java stocké dans le nœud Code du Repository ou enregistré par un cBeanRegister.

Objectif

Le cBean vous permet d'invoquer un bean stocké dans le nœud Code du Repository ou enregistré par un cBeanRegister.

Basic settings

Reference

Sélectionnez cette option afin de référencer un bean Java enregistré par un cBeanRegister.

Dans le champ Id qui apparaît, saisissez l'ID du bean Java.

 New instance

Sélectionnez cette option pour invoquer un bean Java stocké dans le nœud Code du Repository.

Dans le champ Bean class qui apparaît, saisissez le nom de la classe du bean.

Pour plus d'informations concernant la création et l'utilisation de beans Java, consultez le Guide utilisateur de Talend Open Studio for ESB.

 

Specify the method

Cochez cette case pour saisir le nom d'une méthode à inclure dans le bean.

Utilisation

Le cBean peut être utilisé comme composant de début, de milieu ou de fin dans une Route.

Limitation

 

Scénario associé

Pour un scénario associé, consultez :

  • cConvertBodyTo : la section intitulée « Scénario : Convertir le corps d'un fichier XML en org.w3c.dom.Document.class ».
