Composants Processor

Ce chapitre passe en revue les principaux composants présents dans le groupe Processor de la Palette, dans la perspective Mediation de Talend Open Studio for ESB.

La famille Processor regroupe les composants permettant d'effectuer tous types de tâches de traitement des flux de messages, comme le monitoring des messages envoyés ou reçus, la configuration du mode d'échange de messages, le contrôle du temps de livraison, etc.
